Adds an image to the processing queue of the Netra API
Parameters:
Name Type Attributes Default Description
image_url string a valid url of a public image
endpoint string classification model type. Object from ImageDetectionService.endpoints
callback_url string <optional>
 callback url, will override ImageDetectionService.defaultCallbackUrl
threshold number <optional>
 null detection threshold, an integer between 0 and 100, if not specified the default for the model type will be used
query_params object <optional>
 data that will be appended to the query params of the callback url
metadata object <optional>
 data that will be added to a property called "metadata" in the callback JSON
